Features

Everything your operator needs

Built around how worship services actually run — not how software companies think they do.

🖥️

Three Screens at Once

Independently control the Hall, Stage, and Preacher displays from a single operator console. Each screen shows only what it should.

🎼

Lyrics + Chords

Show clean lyrics to the congregation while musicians see chords in their key — with one-click transposition to any key on the fly.

⚡

Special Cues for Musicians

Send instant signals to the stage screen: A Cappella, Drums Only, Repeat, Slower — as brief banners the hall never sees.

📱

Works with Any Equipment

One PC and one projector? That's enough. Musicians can use their phones as the stage screen over Wi-Fi — no extra hardware needed.

📚

Song Library

Organize your congregation's songs with full editing, verse numbering, and automatic transposition. Build setlists in seconds.

🌐

Works Offline

No internet required during services. Everything runs locally. Cloud is only used for optional song library backup.

How it works

Simple for operators, powerful for services

01

Build your setlist

Pick songs from your library, set keys, arrange the order. Takes 5 minutes before service.

02

Run the service

Click through verses and choruses. Each screen updates instantly. Send cues to musicians with one tap.

03

Never lose the flow

Blackout button, quick messages, key changes mid-song — everything you need without breaking the moment.
